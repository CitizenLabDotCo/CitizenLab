Phases API

Update a phase

PATCH api/v1/phases/:id

Parameters

Name Description
phase[project_id] The id of the project this phase belongs to
phase[title_multiloc] The title of the phase in nultiple locales
phase[description_multiloc] The description of the phase in multiple languages. Supports basic HTML.
phase[start_at] The start date of the phase
phase[end_at] The end date of the phase

Response Fields

Name Description
errors[project] Array containing objects with signature {error: 'blank' }
errors[title_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[description_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[start_at] Array containing objects with signature {error: 'blank' }
