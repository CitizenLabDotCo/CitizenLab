Idea Votes API

Create a vote to an idea

POST api/v1/ideas/:idea_id/votes

Parameters

Name Description
vote[user_id] The user id of the user owning the vote. Signed in user by default
vote[mode] one of [up, down]

Response Fields

Name Description
errors[votable] Array containing objects with signature {error: 'blank' }
errors[mode] Array containing objects with signature {error: 'inclusion' }
errors[user] Array containing objects with signature {error: 'blank' }
