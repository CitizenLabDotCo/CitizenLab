Comments API

Create a comment to an idea

POST api/v1/ideas/:idea_id/comments

Parameters

Name Description
comment[author_id] The user id of the user owning the comment. Signed in user by default
comment[body_multiloc] Multi-locale field with the comment body
comment[parent_id] The id of the comment this comment is a response to

Response Fields

Name Description
errors[body_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[author_name] Array containing objects with signature {error: 'blank' }
errors[author] Array containing objects with signature {error: 'blank' }
