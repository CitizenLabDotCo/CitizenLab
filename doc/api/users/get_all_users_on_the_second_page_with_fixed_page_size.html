Users API

Get all users on the second page with fixed page size

GET api/v1/users

Parameters

Name Description
page[number] Page number
page[size] Number of users per page
search Filter by searching in first_name, last_name and email
sort Sort user by 'created_at', '-created_at', 'last_name', '-last_name', 'email', '-email', 'role', '-role'
