Users API

Edit a user

PUT api/v1/users/:id

Parameters

Name Description
user[first_name] User full name
user[last_name] User full name
user[email] E-mail address
user[password] Password
user[locale] Locale. Should be one of the tenants locales
user[avatar] Base64 encoded avatar image
user[roles] Roles array, only allowed when admin
user[gender] Either 'male', 'female' or 'unspecified'
user[birthyear] The year, as an integer, the user was born
user[domicile] Either an exisiting Area id or 'outside', to specify the user does not live in the city
user[education] An integer from 0 to 8 (inclusive), corresponding to the ISCED 2011 standard
user[bio_multiloc] A little text, allowing the user to describe herself. Multiloc and non-html

Response Fields

Name Description
errors[email] Array containing objects with signature {error: 'blank' }
errors[slug] Array containing objects with signature {error: 'invalid' }
errors[first_name] Array containing objects with signature {error: 'blank' }
errors[last_name] Array containing objects with signature {error: 'blank' }
errors[locale] Array containing objects with signature {error: 'inclusion' }
errors[bio_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[gender] Array containing objects with signature {error: 'inclusion' }
errors[birthyear] Array containing objects with signature {error: 'greater_than' | 'greater_than_or_equal_to' | 'equal_to' | 'less_than' | 'less_than_or_equal_to' | 'not_a_number' | 'not_an_integer' | 'odd' | 'even' | 'other_than' }
errors[domicile] Array containing objects with signature {error: 'inclusion' }
errors[education] Array containing objects with signature {error: 'greater_than' | 'greater_than_or_equal_to' | 'equal_to' | 'less_than' | 'less_than_or_equal_to' | 'not_a_number' | 'not_an_integer' | 'odd' | 'even' | 'other_than' }
errors[password] Array containing objects with signature {error: 'wrong_length' | 'too_short' | 'too_long' }
