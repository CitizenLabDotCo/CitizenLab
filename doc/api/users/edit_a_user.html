Users API

Edit a user

PUT api/v1/users/:id

Parameters

Name Description
user[first_name] User full name
user[last_name] User full name
user[email] E-mail address
user[password] Password
user[locale] Locale. Should be one of the tenants locales
user[avatar] Base64 encoded avatar image
user[roles] Roles array, only allowed when admin
user[gender] Either 'male', 'female' or 'unspecified'
user[birthyear] The year, as an integer, the user was born
user[domicile] Either an exisiting Area id or 'outside', to specify the user does not live in the city
user[education] An integer from 0 to 8 (inclusive), corresponding to the ISCED 2011 standard
