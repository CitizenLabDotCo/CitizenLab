Events API

Update a event

PATCH api/v1/events/:id

Parameters

Name Description
event[project_id] The id of the project this event belongs to
event[title_multiloc] The title of the event in nultiple locales
event[description_multiloc] The description of the event in multiple languages. Supports basic HTML.
event[location_multiloc] The location of the event. Textual
event[start_at] The start datetime of the event
event[end_at] The end datetime of the event

Response Fields

Name Description
errors[project] Array containing objects with signature {error: 'blank' }
errors[title_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[description_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[location_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
