Pages API

Updating the page

PATCH api/v1/pages/:id

Parameters

Name Description
page[title_multiloc] The title of the page, as a multiloc string
page[body_multiloc] The content of the page, as a multiloc HTML string
page[slug] The unique slug of the page

Response Fields

Name Description
errors[title_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[body_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[slug] Array containing objects with signature {error: 'invalid' }
