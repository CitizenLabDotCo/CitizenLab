Ideas API

Updating the idea title

PATCH api/v1/ideas/:id

Parameters

Name Description
idea[lab_id] The idea of the lab that hosts the idea
idea[author_id] The user id of the user owning the idea
idea[publication_status] Either ["draft", "published", "closed", "spam"].join(', ')}
idea[title_multiloc] Multi-locale field with the idea title
idea[body_multiloc] Multi-locale field with the idea body
idea[images] Base64 encoded images
idea[files] Multipart form encoded files
