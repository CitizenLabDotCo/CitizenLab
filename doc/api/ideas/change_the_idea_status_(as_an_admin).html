Ideas API

Change the idea status (as an admin)

PATCH api/v1/ideas/:id

Parameters

Name Description
idea[project_id] The idea of the project that hosts the idea
idea[author_id] The user id of the user owning the idea
idea[idea_status_id] The status of the idea, only allowed for admins
idea[publication_status] Either draft, published, closed, spam
idea[title_multiloc] Multi-locale field with the idea title
idea[body_multiloc] Multi-locale field with the idea body
idea[topic_ids] Array of ids of the associated topics
idea[area_ids] Array of ids of the associated areas
idea[location_point_geojson] A GeoJSON point that situates the location the idea applies to
idea[location_description] A human readable description of the location the idea applies to

Response Fields

Name Description
errors[title_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[body_multiloc] Array containing objects with signature {error: 'blank' | 'unsupported_locales' }
errors[publication_status] Array containing objects with signature {error: 'inclusion' }
errors[author] Array containing objects with signature {error: 'blank' }
errors[author_name] Array containing objects with signature {error: 'blank' }
errors[idea_status] Array containing objects with signature {error: 'blank' }
errors[slug] Array containing objects with signature {error: 'invalid' }
