Ideas API

Creating a published idea

POST api/v1/ideas

Parameters

Name Description
idea[project_id] The idea of the project that hosts the idea
idea[author_id] The user id of the user owning the idea
idea[publication_status] Password
idea[title_multiloc] Multi-locale field with the idea title
idea[body_multiloc] Multi-locale field with the idea body
idea[images] Array with base64 encoded images
idea[files] Array with base64 encoded files
idea[topic_ids] Array of ids of the associated topics
idea[area_ids] Array of ids of the associated areas
