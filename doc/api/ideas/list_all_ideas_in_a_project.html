Ideas API

List all ideas in a project

GET api/v1/ideas

Parameters

Name Description
topics Filter by topics (AND)
areas Filter by areas (AND)
project Filter by project
author Filter by author (user id)
idea_status Filter by status (idea status id)
search Filter by searching in title, body and author name
sort Either 'new', '-new', 'trending', '-trending', 'popular', '-popular', 'author_name', '-author_name', 'upvotes_count', '-upvotes_count', 'downvotes_count', '-downvotes_count', 'status', '-status'
publication_status Return only ideas with the specified publication status; returns all pusblished ideas by default
