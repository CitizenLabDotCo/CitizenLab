Ideas API

List all ideas with all given areas

GET api/v1/ideas

Parameters

Name Description
topics Filter by topics (AND)
areas Filter by areas (AND)
project Filter by project
author Filter by author (user id)
search Filter by searching in title, body and author name
sort Either 'new', '-new', 'trending', '-trending', 'popular' or '-popular'
