Ideas API

List all ideas with search string

GET api/v1/ideas

Parameters

Name Description
topics Filter by topics (AND)
areas Filter by areas (AND)
lab Filter by lab
author Filter by author (user id)
search Filter by searching in title, body and author name

Content-Type: application/json

GET api/v1/ideas?search=uniqque
